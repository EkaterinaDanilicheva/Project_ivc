CLNT_CLNT_ID
NUMBER
идентификатор клиента (обязательно к заполнению)
SUBS_SUBS_ID
NUMBER
идентификатор абонента относящегося к клиенту (обязательно к заполнению)
IP_START
VARCHAR2(32)
IP начало диапазона в виде mapped IPv4 в IPv6 (формат записи HEX в нижнем регистре без разделителей) (обязательно к заполнению)
IP_END
VARCHAR2(32)
IP конец диапазона в виде mapped IPv4 в IPv6 (формат записи HEX в нижнем регистре без разделителей) (обязательно к заполнению)
DATE_START
DATE
начало выделения диапазона (дата и время) (обязательно к заполнению)
DATE_END
DATE
конец выделения диапазона (дата и время) (обязательно к заполнению)
REGION_ID
NUMBER
регион или провайдер, (может быть пустым)
